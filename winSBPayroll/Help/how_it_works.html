Soft Books Payroll - How it Works

Home

  1. Soft Books Payroll is a low-cost payroll solution that supports an unlimited number of employees and users, with support for tracking benefit accruals and multiple deduction types, as well as providing compliance reporting tools. The system comprises of integrated modules in all aspects of Payroll functions including management of pension disbursements.
  2. The software is preferably configured on server to be accessed by various authorized users from their respective workstations.
  3. Soft Books Payroll System is a multi-user Client/Server.
  4. Tax is a dynamic area in Kenya. The changes that Government implement on tax policies have far reaching implications on business every year and as a result there is need for a dynamic system to keep abreast of them.
  5. The objective of the system is to assist businesses manage their growth process, finance, investment and management issues generally adding professional value to the business. The Soft Books Payroll system provides the ultimate solution for organizations and Higher Education Institutions with a diverse range from small to large employee base. It offers support for accountants representing multiple clients or can be used directly by small and mid-sized businesses.
  6. The Soft Books Payroll system has been developed using the latest software development tools to ensure that it meets all the requirements a user could want from a payroll (either in-house, hosted or outsourced).
